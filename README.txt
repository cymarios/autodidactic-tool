Autodidactic Learning Tool
==========================

HOW TO RUN:

1. BACKEND:
------------
Open Anaconda Prompt:

> conda activate autodidactic
> cd backend
> python app.py

This starts the learner server on http://127.0.0.1:5000

2. FRONTEND:
-------------
Double-click frontend/index.html to open it in your browser.

Type in binary examples (e.g., 1?0) and label (1 or 0).  
Click Submit to update the hypothesis.  
Click Reset to start over.

3. JUPYTER NOTEBOOK (Optional):
--------------------------------
If you'd like to experiment with the algorithm in Python:

> cd notebooks
> jupyter notebook Autodidacticproject.ipynb
